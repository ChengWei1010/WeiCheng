NSYSU HERO

Game UI Design

Overview

This Unity 2-Dimentional game was made in a 2 days hackathon in Department of Information Management at National Sun Yat-sun University.

Role

Graphic Designer

Tools

Adobe Illustrator / Adobe Photoshop / Unity

Story

NSYSU HERO is based on the geographical background of my university, National Sun Yat-sun University.

Soma part of the mountainous campus are the natural habitat of Formosan macaque, a protected species in Taiwan.

However, sometimes the macaques can be pretty aggressive and attack students. Thus, campus safeguards use airgun to scare off the monkeys preventing them from attacking human, and then bring them back to their home to protect them.

We created NSYSU HERO to practice the skills of using Unity and designing user interface of games.

Game Scenes
1 - Home Page

The home page let user to go to two pages.
How to play takes user to the game instruction page, and the Start will start a new game.

2 - Instruction Page

The instruction page use photos and text to explain the controls and rules about how to play this game.
The back button take user back to the home page.

03 - Game Scene

This is the main page for the game.
The character should avoid bumping to the bus and have to jump on to the building.
The Donut indicates the life of the character.
The Score indicates the current score of this game.
The End indicates end this game and show the result.

Conclusion

I really enjoyed drawing the game scenes and UI elements for this game. I also learned how to communicated with game developer in a gaming project. I not only practiced team work skills, but also practiced my design skills in this project.
